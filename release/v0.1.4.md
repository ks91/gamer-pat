# GAMER PAT Prompt-Agent v0.1.4

## Summary
- Improves the PDF annotation extraction helper used to let PAT read reviewer feedback from annotated PDFs.
- Fixes several real-world extraction failures found in commented academic PDFs.
- Updates prompt-agent package version metadata to `0.1.4`.

## Included in This Release
- Improved English word spacing in extracted PDF targets and context.
- Preserved Japanese text and comments more reliably.
- Added support for PDF text annotations in addition to highlights.
- Extracted annotations stored inside expanded PDF object streams.
- Fixed page mapping by following the PDF page tree order instead of object-number order.
- Sanitized invalid XML control characters emitted by `pdftotext -bbox-layout`.
- Added regression tests for spacing, Japanese text handling, text annotations, object streams, page-tree ordering, and invalid XML characters.
