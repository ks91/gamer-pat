# GAMER PAT Prompt-Agent v0.1.2

## Summary
- Adds a distributable WBS MBA thesis LaTeX template package.
- Adds a helper script for extracting annotations from commented PDFs.
- Finalizes tested platform guidance carried forward after v0.1.1.

## Included in This Release
- Updated prompt-agent package version metadata to `0.1.2`.
- Confirmed Raspberry Pi OS and Chrome OS PDF viewer guidance as tested.
- Added `templates/wbs-mba-thesis/`:
  - `wbsmba.cls`
  - `wbsapalike.bst`
  - `wbsplain.bst`
  - sample `.tex`, `.bib`, and `.pdf` files
  - `bibliography-guide.md`
- Standardized sample layout so the WBS templates can be compiled directly from the package root.
- Added `scripts/extract_pdf_annotations.py` and `scripts/README.md`.
- Added `.gitignore` for local logs and TeX build artifacts.
